## المشكلة

كثير من الموظفين والعمال في المملكة العربية السعودية لا يعرفون حقوقهم القانونية، ويجدون صعوبة بالغة في فهم نظام العمل بسبب:

- تعقيد اللغة القانونية وصعوبة تفسيرها
- الحاجة إلى محامٍ لأبسط الاستفسارات
- غياب أداة عربية موثوقة ومتاحة للجميع

## الحل

بيّن هو مساعد قانوني ذكي يدمج أحدث تقنيات الذكاء الاصطناعي للإجابة الفورية على أسئلة نظام العمل بلغة بسيطة، وصياغة الخطابات الرسمية، وشرح المواد القانونية مع ذكر المرجع.

---

## الميزات الأساسية

| الميزة | الوصف |
|--------|-------|
| بحث ذكي في نظام العمل | يفسر السؤال، يسترد المواد الأكثر صلة، ويولّد إجابة واضحة مع رقم المادة |
| صياغة خطابات رسمية | ينشئ خطابات مهنية (شكوى، طلب، استقالة) مع إمكانية التعديل عبر الحوار |
| شرح المواد القانونية | يدعم البحث بالوصف أو برقم المادة، ويعرض النص الأصلي مع شرح مبسط |
| أسئلة متابعة ذكية | يحتفظ بسياق المحادثة للإجابة على استفسارات متسلسلة |
| واجهة عربية متجاوبة | تعمل على جميع الأجهزة مع دعم كامل للغة العربية (RTL) |

## التقنيات المستخدمة

| المكون | التقنية |
|--------|---------|
| نموذج الذكاء الاصطناعي | Nuha-2.0 — LLM مخصص للقانون السعودي |
| خوارزمية RAG | مطابقة كلمات مفتاحية محسّنة (TF-IDF-like) تعمل داخل المتصفح |
| قاعدة البيانات | JSON يحتوي على 245 مادة من نظام العمل السعودي |
| الخادم الوسيط | Node.js + Express |
| الواجهة الأمامية | HTML5، CSS3، JavaScript (ES6) — بدون أطر خارجية ثقيلة |
| الاستضافة | Railway + GitHub |

## متطلبات التشغيل

- [Node.js](https://nodejs.org/) v18 أو أحدث
- مفتاح API لنموذج Nuha-2.0
- حساب على [Railway](https://railway.app/) للاستضافة *(اختياري للتشغيل المحلي)*

---

## تشغيل المشروع محلياً

### 1. تثبيت الاعتماديات
```bash
npm install
```

### 2. إعداد متغيرات البيئة
أنشئ ملف `.env` في جذر المشروع وأضف:

```env
API_KEY=your_nuha_api_key_here
API_URL=https://api.nuha.ai/v1/chat/completions
```

### 3. تشغيل الخادم
```bash
node server.js
```

افتح المتصفح على: [http://localhost:3000](http://localhost:3000)

---

## النشر على Railway

1. في Railway، أنشئ مشروعاً جديداً واختر **Deploy from GitHub repo**
2. اربط المستودع
3. أضف متغيري البيئة `API_KEY` و `API_URL` في إعدادات Railway
4. سيتم النشر تلقائياً وستحصل على رابط مثل: `https://bayen.up.railway.app`

> **ملاحظة:** تأكد من وجود ملف `labor_law_cleaned.json` في الجذر لأن الواجهة تحمله عبر `fetch`.

## واجهة API

### `POST /api/chat`

**الطلب:**
```json
{
  "messages": [
    { "role": "system", "content": "أنت محلل قانوني متخصص في نظام العمل السعودي..." },
    { "role": "user", "content": "ما حقوقي إذا أنهى صاحب العمل عقدي بدون سبب؟" }
  ]
}
```

**الرد:**
```json
{
  "choices": [
    {
      "message": {
        "content": "أولاً: يحق لك مكافأة نهاية الخدمة وفق المادة 84، وثانياً..."
      }
    }
  ]
}
```
